# Крутой перевод и тд
- Более улучшенный русский перевод интерфейса приложения XPPen
- Чтобы использовать его, замените файл на language.ini в AppData/Roaming/XPPen

## Что было изменено

- Использование нормальных терминов и сленга
- Отсутствие машинного перевода (они будто кинули в гугл переводчик всё)
- Исправлены опечатки и неточности  

## Основные исправления

| До | После |
|---|---|
| батарея | Батарея |
| Быстрые клавиши | Сочетания клавиш |
| Колесо управления | Кольцо |
| Щелчок левой кнопкой мыши | Левый щелчок |
| Инструмент диагностики | Средство диагностики |
| Прикосниться | Сенсорное управление |

## Файлы

- `language.ini` - раздел [Russian] с полным переводом
